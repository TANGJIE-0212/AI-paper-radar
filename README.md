# AI Paper Radar

A weekly AI paper radar that tracks HuggingFace's top papers and delivers product-lens analysis. Fully automated via GitHub Actions + OpenClaw.

**Live site**: [https://tangjie-0212.github.io/AI-paper-radar/](https://tangjie-0212.github.io/AI-paper-radar/)

## How It Works

```
Every Sunday 09:00 Beijing         09:30 Beijing              Auto on push
┌──────────────────┐        ┌───────────────────┐       ┌──────────────┐
│  GitHub Actions   │        │     OpenClaw       │       │ GitHub Pages │
│  fetch_papers.py  │ ────▶  │  Deep analysis     │ ────▶ │  Auto deploy │
│  → papers.json    │        │  → data.js + push  │       │              │
└──────────────────┘        └───────────────────┘       └──────────────┘
```

1. **GitHub Actions** fetches the weekly HuggingFace paper leaderboard → `scripts/papers.json`
2. **OpenClaw** reads `papers.json`, performs deep analysis per `SKILL.md`, generates `assets/data.js`, and pushes
3. **GitHub Pages** auto-deploys on push to `assets/`

## Project Structure

```text
.
├── .github/workflows/weekly-update.yml   # Cron: fetch papers + deploy on push
├── assets/
│   ├── data.js          # Generated by OpenClaw (deep analysis)
│   └── index.html       # Static UI (bilingual, dark theme)
├── scripts/
│   ├── fetch_papers.py  # HuggingFace scraper
│   ├── build_data_js.py # Fallback template generator (not used in automation)
│   └── papers.json      # Raw paper data
└── SKILL.md             # OpenClaw task definition
```
